# Exercices JavaScript : Applications Web simples

Ce dépôt contient quatre petites applications web développées en HTML, CSS et JavaScript pour pratiquer les bases de la programmation côté client. Chaque exercice correspond à un fichier HTML autonome.

## Fichiers inclus

1. **`permutation.html`** – Permutation de texte  
   - Deux zones de texte et un bouton.  
   - Clic sur le bouton pour échanger le contenu des deux champs.
   - 
2. **`imc.html`** – Calculateur d'IMC (Indice de Masse Corporelle)  
   - Saisie du poids (kg) et de la taille (m).  
   - Calcule l'IMC et affiche la catégorie correspondante (maigreur, corpulence normale, surpoids, obésité modérée, sévère ou morbide).  
   - Accepte les virgules comme séparateur décimal.

3. **`calculatrice-simple.html`** – Calculatrice simple  
   - Deux champs de saisie numérique et quatre boutons d'opérations (+, -, *, /).  
   - Affiche le résultat dans un champ en lecture seule.  
   - Gère la division par zéro.
     
4. **`calculatrice-avancée.html`** – Calculatrice scientifique  
   - Interface inspirée des calculatrices classiques avec touches : chiffres, opérateurs (+, -, *, /, %), parenthèses, fonctions scientifiques (sin, cos, tan, ln, log, √, exp, x², puissance, constante e).  
   - Affiche l’expression saisie et le résultat après évaluation.  
   - Gère les erreurs de syntaxe.

## Utilisation

Chaque fichier est indépendant. Pour l’utiliser, ouvrez-le simplement dans un navigateur web moderne (Chrome, Firefox, Edge, etc.). Aucune installation ni serveur n’est nécessaire.

## Technologies utilisées

- HTML5  
- CSS3 (mise en forme simple)  
- JavaScript (ES6) pour la logique interactive  

## Objectifs pédagogiques

- Manipulation du DOM (récupération et modification d’éléments).  
- Gestion des événements (clics sur boutons).  
- Conversion et validation de données utilisateur.  
- Implémentation d’algorithmes simples (permutation, calculs arithmétiques, IMC).  
- Construction d’une interface utilisateur réactive.

## Améliorations possibles

- Ajouter plus de fonctions à la calculatrice avancée (factorielle, inverse, etc.).  
- Améliorer le style CSS avec une mise en page plus moderne.  
- Rendre les calculateurs accessibles (attributs ARIA, navigation au clavier).
